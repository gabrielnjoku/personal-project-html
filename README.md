# Personal Profile Website - HTML Individual Assessment

## Project Overview
This project is a multi-page personal profile website built as an individual assessment for the **EnuguSME Turing Tech Training Program (ETTP)**. The primary goal is to demonstrate a core understanding of HTML fundamentals, including semantic structuring, table creation, form building, and internal page navigation.

## Features
* **Sidebar Navigation**: A navigation panel built with an unordered list and anchor links for same-page scrolling.
* **Semantic Structure**: Developed using modern HTML5 semantic elements to structure the page meaningfully.
* **Home Section**: A welcome section introducing the author with a photo and introductory paragraph.
* **About Section**: A biography highlighting professional experience as an IT Support Specialist and Multi-Cloud Engineer.
* **Education Table**: A structured data table showcasing academic history. It includes `<thead>`, `<tbody>`, and at least three rows of data.
* **Hobbies List**: An unordered list displaying at least five hobbies and interests.
* **Contact Form**: A functional form layout with fields for Full Name, Email Address, and Message.

## Technologies Used
* **HTML5 Only**: This project uses no CSS or styling tags, focusing strictly on HTML structure.

## File Structure
* `index.html`: The single HTML file containing all sections and navigation.
* `images`: The folder containing the profile photo of the author.
